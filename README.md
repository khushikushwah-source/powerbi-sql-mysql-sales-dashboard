📊 End-to-End Power BI Dashboard Project (SQL Server & MySQL)

📌 Project Overview

This project demonstrates a complete end-to-end data analytics workflow using SQL Server, MySQL, and Power BI. The objective of the project is to transform raw data into meaningful insights by building an interactive Power BI dashboard with KPIs and visual analytics.
The project simulates a real-world data pipeline including data extraction, transformation, cleaning, modeling, and visualization.

🛠 Tech Stack

• Power BI Desktop
• SQL Server
• MySQL Server
• MySQL Workbench
• Power Query Editor
• DAX (Data Analysis Expressions)
• MySQL Connector

⚙️ Project Workflow

1️⃣ Database Environment Setup

• Installed MySQL Server and MySQL Workbench
• Set up the Test Environment database
• Imported dataset into SQL Server

2️⃣ Understanding Dataset & Requirements

• Analyzed dataset structure
• Identified reporting requirements
• Understood relationships between tables

3️⃣ Data Preparation using SQL Server

• Applied LEFT JOIN to combine multiple tables
• Structured the data for reporting
• Prepared clean dataset for Power BI analysis

4️⃣ Importing Data into Power BI

• Connected Power BI Desktop with SQL Server
• Imported dataset into Power BI
• Used Power Query Editor for additional transformation

5️⃣ Dashboard Development

Created interactive Power BI dashboard pages including:

• KPI Cards
• Business Metrics
• Trend Analysis
• Performance Insights

Developed DAX Measures & KPIs for:

• Page 1 dashboard metrics
• Page 2 dashboard metrics

6️⃣ Production Environment Setup

• Migrated dataset from Test Environment to Production Environment
• Performed data cleaning using SQL

7️⃣ MySQL Database Integration

• Installed MySQL Connector
• Imported data into MySQL Database
• Created equivalent SQL queries in MySQL Workbench to populate new tables

8️⃣ Power BI Report Deployment

• Created Power BI Workspace
• Published SQL Server data source report
• Updated data source using Advanced Editor in Power Query

Performed:

• Data validation
• Report verification
• Final dashboard publishing connected to MySQL database
